## 📖 Overview

**Amalgam V2** is a fully **rebranded and enhanced version** of the original project.
The goal of this version is to modernize the interface, improve performance, and expand combat-related systems while maintaining a clean and optimized codebase.

This release focuses on:

* ⚡ Performance improvements
* 🎨 Modern UI/UX redesign
* 🧠 Smarter combat systems
* 🧹 Cleaner code structure

---

## ✨ Features

### 🎨 Modern UI

* High-quality **dark themed interface**
* **Alternating row indicators** for better readability
* Improved layout for binds, indicators, and menus
* Smoother rendering and UI performance

### ⚔️ Combat Improvements

**Doubletap (DT)**

* Reworked recharge logic
* Faster activation response
* Better tick synchronization

**Anti-Aim (AA)**

* Improved **edge detection**
* New **jitter modes**

  * Switch
  * Random

**Desync System**

* Improved yaw desync logic
* Better defensive behavior

---

## 🔄 Rebranding

This project has undergone a **complete identity overhaul**.

Changes include:

* Removal of all original identifiers
* Updated naming conventions
* Cleaned project structure
* Full transition to **Amalgam V2**

## 📥 Building

1. Clone the repository

```
git clone https://github.com/DEVXeno-ops/Amalgam-v2.git
```

2. Open the solution file

```
Amalgam-v2.sln
```

3. Build using **Visual Studio 2022**

Configuration:

```
Release | x64
```
