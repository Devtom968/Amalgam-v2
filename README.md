  ### 🚀 Rebranded & Improved

  This is a modernized version of the original cheat, now renamed to **Amalgam V2**.
  Key improvements include a sleek and readable GUI, enhanced Doubletap logic, and advanced Anti-Aim features.

  ---

  ## ✨ Features in V2

  - **UI Modernization**: Sleek dark theme with alternating row highlights in the Binds window for better readability.
  - **Enhanced Doubletap (DT)**: Improved recharge logic for faster and more consistent shots.
  - **Advanced Anti-Aim (AA)**: 
    - **Improved Edge Detection**: Better wall detection for stealthy movement.
    - **Switch & Random Jitter**: More unpredictable yaw movement.
    - **Desync Logic**: Enhanced desyncing while choking packets.
  - **Rebranding**: Complete removal of old references, replaced with **Amalgam V2**.
